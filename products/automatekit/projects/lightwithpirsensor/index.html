            # Light bulb on/off with PIR sensor

            # Introduction

            In this project, using KiMu's Automate Advanced Kit, we are going to build a system that uses a relay switch and a PIR sensor to automatically turn on/off the light bulb based on the motion detection.

            # Key skills/Learnings

            1. Problem Solving
            2. Logical Thinking
            3. Working of relay switch & PIR sensor

            # Assemble the System

            Note: All activities which involve main voltages (voltage greater than 50V) should be performed under adult supervision.

            Step 1: Connect the light bulb to the power plug with a wire(black).
            Step 2: Connect one end of the relay to the light bulb with wire(red) and the otherend to the power plug.
            Step 3: Connect PIR sensor to U2 connector of the sensor board.
