            # Automatic Fan On/Off using NTC Sensor

            # Introduction

            Sometimes we do feel lazy to turn the fan on, when the conditions are hot and turn it off, when the conditions are cold. Wouldn't it be nice if there is an automated system which performs this task? With KiMu's Advanced Automate Kit, we are going to build a system to automatically turn on/off the fan based on the temperature using NTC Sensor. The project has wide range of applications like Cooling fans for laptops and computers, energy conservation applications.

            # Key skills/Learnings

            1. Problem Solving
            2. Logical Thinking
            3. Working of NTC sensor
