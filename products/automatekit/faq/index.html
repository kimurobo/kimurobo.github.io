                          # FAQs on Automate Kit

                          # 1. How to use debug feature in the Automate App?

                          Debugging is the process of identifying and removing errors/problems from the program.

                          • Go to Debug page in the Automate App and select how you have connected to the control unit. (Cable or WIFI)

                          • From the drop down select how you want to get data

                            • on button click – get debug data when “on click” button is clicked.
                            • Time delay – get debug data every X seconds. (say 1 second)
                            • On step change – get debug data whenever step changes.

                          • Now click on “START” button.

                            • If you have selected “on button click” option in the dropdown, you have to click “on click” button to get data in the table
                            • If you have selected “time delay” or “on step change” data will automatically be displayed in the table.

                          • Click “STOP” button to stop receiving data.

                          • Click “CLEAN” button to clean the data displayed in the table.

                          • From debug screen you can get the readings of

                            • Time
                            • Present step
                            • Previous step
                            • Step change reason
                            • Voltage input 1
                            • Voltage input 2
                            • Digital input 1
                            • Digital input 2
                            • Distance sensor
                            • Input voltage

                          # 2. How to use steps feature in the Automate App?

                          All the devices connected to the control unit can be configured to behave differently in different step.

                          • There are 16 steps.
                          • At a given time, control card will only be in any one of the 16 steps.
                          • After the assigned tasks in a step are completed, you can either repeat the same step or jump to a different step.
                          • Step 1 is the starting step for any program.
                          • Each time when the control unit is turned on or new program is uploaded, there will be a delay of 5 seconds. After that, program will start from step 1 by default.

                          Refer to the App Manual Provided in the kit, for more information.

                          # 5. How do voltage input sensors provided in the Automate kit work?

                          The voltage being sensed gradually changes according to the parameter it is sensing.

                          • Light sensor – as the light falling on the sensor increases, voltage read by the voltage input increases.
                          • Moisture sensor – as the moisture content increases, voltage read by the voltage input increases.
                          • Temperature sensor (NTC) – as the temperature increases, voltage read by the voltage input increases.
                          • The remaining sensors (IR, PIR & Sound sensors) work a bit differently. They can give only two fixed voltages. A lower voltage if the parameter is not present and a higher voltage if the parameter is present.

                          The voltage given by a sensor can be found by connecting that sensor to the Automate Control Unit and using the debug feature in Automate App.

                          # 6. How does IR Sensor work and how to calibrate it ?

                          If the IR receiver detects some IR rays, voltage input will detect X volts. If the IR receiver didn’t detect IR rays, voltage input will detect Y volts. How much of the IR rays will get reflected after hitting the object depends on the type of material. Some materials will completely reflect, some will completely absorb the IR rays and some will reflect a part of it and absorb the remaining. So, some testing has to be done for each type of material you are going to use.

                          To calibrate IR sensor for black/white color detection,

                          • Mount the IR sensor about 1 to 2 inches above a black surface and connect it to the Automate Control Unit.
                          • Slowly turn the potentiometer (blue cube) using a screwdriver in any direction, to set the reference voltage for the black surface.
                          • The reference voltage can be found using debug feature in the Automate App.

                          # 7. How to calibrate Sound Sensor ?

                          To Calibrate Sound Sensor,

                          • Connect the Sound Sensor to the Automate Control unit via sensors board.
                          • Make a sound closer to the sensor and use the debug feature in the Automate App to get the voltage input from the sensor.
                          • Slowly turn the potentiometer (blue cube) using a screwdriver in any direction, to set the reference voltage for the reference sound.
                          • If the sound being sensed is lower than reference sound, voltage input will sense X volts. If the sound being sensed is higher than reference sound, voltage input will sense Y volts.

                          # 12. What devices can be connected to Digital Input port in the Automate Control Unit?

                          Limit Switches can be connected to Digital Input ports.

                          Digital inputs are used to detect the state of a switch (on/off). When the switch is on, digital input will detect 5v. When the switch is off, digital input will detect 0v. If the voltage sensed by digital input is the same as what you selected, then the control card will jump to the selected step
